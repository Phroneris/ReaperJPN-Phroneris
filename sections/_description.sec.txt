;【はじめに】
; ・OS標準のメモ帳ではなく、ある程度本格的なテキストエディタで編集することを強く推奨します。
;   作者はNotepad++を使っています。お勧めのフリーソフトです。
; ・ある程度本格的なテキストエディタでは、ファイルの言語に「MS INIファイル」か何かを選択すると、
;   多分シンタックスハイライトが効いて編集しやすくなります。
;
;【*.ReaperLangPackの記法】
; ・"[common]" などの行はセクション名です。
; ・";" で始まる行は未翻訳部です。翻訳後は ";" を消してください。
;   もっと言うと、";" で始まる行はコメントアウトなので何を書いても影響しません。この解説部もそうです。
; ・";^" で始まる行は未翻訳部かつオプションです。翻訳後は ";^" を消してください。
;   最初の [common] セクションで既に出てきているので、改めて別の訳を当てたい場合のみ編集します。
; ・セクション内に "5CA1E00000000000=幅 高さ" という記述を追加すると、
;   該当のウィンドウサイズを変更できます。
;   高さは省略できます。以下の例なら幅1.2倍です。
;     [DLG_218] ; IDD_LOCKSETTINGS
;     5CA1E00000000000=1.2
;
;【このパッチ独自の記法】
; ・行中の「(タブ);」以下は備考コメント、「(タブ);;」以下は問題点コメント、
;   「;/」で始まる行は意図的な無効化記述として使っています。
; ・アクション行中の「 ▲▼ 」以降は原文を併記しています。
;
